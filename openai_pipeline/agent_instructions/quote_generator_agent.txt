  - 500-999 units: 10% discount
  - 1000+ units: 15% discount

  Calculate:
  - volume_discount_percent = tier percentage
  - discount_amount = unit_price x quantity x (volume_discount_percent / 100)
  - unit_price_final = unit_price x (1 - volume_discount_percent / 100)
  - line_total = bom_quantity x unit_price_final

  Round all values to 2 decimal places.

  STEP 4: CALCULATE REQUIRED ITEMS TOTALS

  Sum all required line_totals:
  - subtotal = sum of (quantity x unit_price) for all required items

  Calculate total discounts applied:
  - volume_discounts_applied = sum of all discount_amounts

  Calculate subtotal after discounts:
  - subtotal_after_discounts = subtotal - volume_discounts_applied

  Calculate tax (8%):
  - tax_amount = subtotal_after_discounts x 0.08

  Calculate shipping for required items only:
  - If subtotal_after_discounts < $500: shipping = $15.00
  - If subtotal_after_discounts >= $500: shipping = $0.00

  Calculate grand total for required items:
  - grand_total_required = subtotal_after_discounts + tax_amount + shipping

  STEP 5: CALCULATE OPTIONAL ADD-ONS TOTALS

  If cross-sell add-ons exist:

  Calculate add-ons subtotal after discounts:
  - Apply volume discounts to add-on quantities
  - add_ons_subtotal_after_discounts = sum of add-on line_totals

  Calculate tax on add-ons:
  - add_ons_tax = add_ons_subtotal_after_discounts x 0.08

  Calculate shipping for combined order:
  - combined_subtotal = subtotal_after_discounts + add_ons_subtotal_after_discounts
  - If combined_subtotal < $500: combined_shipping = $15.00
  - If combined_subtotal >= $500: combined_shipping = $0.00

  Calculate add-ons total:
  - add_ons_grand_total = add_ons_subtotal_after_discounts + add_ons_tax

  Calculate combined grand total:
  - combined_grand_total = grand_total_required + add_ons_grand_total
